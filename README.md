# Trip Enhancer

Trip Enhancer is a Java-based travel planning application that:

- Fetches tourist attractions using Geoapify API
- Filters and ranks attractions
- Removes duplicate attractions
- Converts user location names into coordinates
- Generates an optimized travel route

## Features

- Attraction Discovery
- Attraction Ranking
- Current Location Geocoding
- Route Optimization
- User Attraction Selection

## Technologies Used

- Java
- Geoapify API
- Gson
- HTTP Client API
